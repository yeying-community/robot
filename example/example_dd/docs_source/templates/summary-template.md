# 群聊总结模板

群聊总结默认使用以下结构：

## 结论

- 先给一句话结论

## 背景

- 说明讨论主题和上下文

## 关键结论

- 列 2~5 条关键结论

## 待办

- 待办事项
- 负责人建议（如果明确）

## 风险 / 待确认

- 风险点
- 仍需确认的点

## 来源

- 说明总结基于哪段讨论 / 哪些文档
