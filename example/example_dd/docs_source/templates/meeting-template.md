# 会议预览模板

会议 / 日程预览默认使用以下结构：

## 主题

- 会议主题

## 时间

- 日期 / 时间

## 组织者

- 默认谁发起，谁组织；若不是，需要明确写出

## 参与人建议

- 建议参与人列表

## 会议形式

- 线上会议 / 线下地点 / 是否需要会前材料

## 会议目标

- 这次会议要解决什么

## 待确认项

- 时间是否已确认
- 参与人是否已确认
- 是否需要会前材料
