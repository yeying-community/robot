# 研发群常见问题 FAQ

## 如何申请项目权限？

先查看项目启动说明和权限申请文档，再联系对应导师审批。机器人只能做文档导航和草稿生成，不会直接开权限。

## 文档沉淀应该怎么做？

先让机器人把讨论整理为 Wiki 草稿，确认后再发布到公司 Wiki。

## 讨论里发现 Bug 怎么办？

可以让机器人生成 GitHub Issue 草稿，确认标题、描述和优先级后再提交。
